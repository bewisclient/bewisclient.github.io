Description
Bewisclient is a Minecraft fabric mod, which has many features that clients like Badlion Client or Lunar Client, but it can be added to all fabric modpacks. It does not boost performance, but its compatible with sodium, so you can get a decent boost. With Iris you can play with shaders.
Features
Fullbright
Shulkerbox Tooltip
Custom Blockhit (Outline)
More Widgets on the Screen (for Example Coordinates, Biome, etc.)
Better Visibility under Fluids (lava, water, powdersnow, etc) and in the nether
Pumpkin Overlay Disabled
Held Item Infos
Mod menu
Cosmetics
Zoom
Scoreboard Size
Macros (Executable Command Chains)
Cleaner Debug menu
